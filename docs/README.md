# Docs

This folder contains extra documentation for `trello-mcp`.

- `setup.md` – environment variables and running the server (SSE recommended)
- `tools.md` – tool catalog and payload examples
- `trello-api-key.md` – how to get a Trello API key and token
- `postman-howto.md` – how to use the provided Postman collections
- `claude-code-howto.md` – how to use this MCP server from Claude Code
